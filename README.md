# CSCI110_Proj1_BankingSystem
Lab: Banking System Application

This project is a C++ banking application that allows users to log in, view their account balance, deposit funds, withdraw funds, and log out.

The program uses a custom Credentials structure to store user information including name, username, password, and account balance. User data is loaded from a file when the program starts and saved back to the file after account changes are made.

The application uses functions to organize different banking operations, including authentication, account menus, deposits, withdrawals, balance display, and file management.

Concepts practiced:

Structures (struct)
Functions and passing by reference
Arrays of structures
File input/output (fstream)
User authentication logic
While loops and menus
Switch statements
Data persistence
